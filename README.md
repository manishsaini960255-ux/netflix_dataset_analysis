# Netflix Dataset Analysis

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix dataset to uncover trends, patterns, and insights related to Netflix content. The analysis focuses on understanding content distribution, ratings, genres, release trends, and country-wise content availability.

## Objectives

* Understand the structure of the Netflix dataset
* Clean and preprocess the data
* Handle missing values and duplicates
* Analyze Movies and TV Shows distribution
* Explore content ratings and genres
* Identify trends across release years
* Generate meaningful visualizations and business insights

## Dataset Information

The dataset contains information about Netflix titles, including:

* Title
* Type (Movie / TV Show)
* Director
* Cast
* Country
* Release Year
* Rating
* Duration
* Genre
* Date Added

## Exploratory Data Analysis

The following analyses were performed:

* Data Inspection
* Missing Value Analysis
* Duplicate Value Detection
* Content Type Distribution
* Rating Analysis
* Country-wise Content Analysis
* Genre Analysis
* Release Year Trends
* Data Visualization

## Key Insights

* Movies significantly outnumber TV Shows on Netflix.
* Netflix content has grown rapidly over the years.
* A few countries contribute the majority of Netflix content.
* Certain ratings dominate the platform's content library.
* Drama and International content are among the most common genres.
